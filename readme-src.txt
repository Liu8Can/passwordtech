COMPILING PASSWORD TECH

  If you want to build the program yourself from the source code, you will need

    - Embarcadero C++Builder 10.4 Update 2 or a later version
    - [optional] Inno Setup to create the setup of Password Tech (you should
	  use the file PwTech.iss for this purpose)
